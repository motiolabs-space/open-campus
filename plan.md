# Konsep Open Source Campus Network (OSCN)

## “LinkedIn + LMS + Academic Impact Platform” untuk Ekosistem Kampus Indonesia

Open Source Campus Network adalah platform open source berbasis komunitas yang dapat diinstall mandiri oleh setiap kampus, namun tetap terhubung dalam sebuah federasi data dan kolaborasi nasional.

Platform ini bukan hanya SIM Akademik atau dashboard pelaporan IKU/PT semata, tetapi menjadi:

* social collaboration platform kampus,
* ecosystem platform antara kampus–industri–masyarakat,
* knowledge & portfolio platform,
* evidence engine untuk IKU PT,
* serta impact monitoring platform.

Konsep utamanya:

> Aktivitas nyata civitas akademika otomatis menghasilkan evidence dan capaian IKU tanpa terasa seperti “mengisi laporan”.
